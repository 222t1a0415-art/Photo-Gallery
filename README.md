# 📸 My Photo Gallery

An interactive photo gallery built with **HTML, CSS, and JavaScript**.  
It features a lightbox with animations, slideshow controls, shuffle mode, keyboard navigation, and a progress bar — all wrapped in a responsive design.

---

## ✨ Features
- 🖼️ Lightbox view with smooth fade-in animations  
- ⏩ Slideshow mode with adjustable speed (2s, 3s, 5s)  
- 🔀 Shuffle option for random image display  
- ⌨️ Keyboard navigation (←, →, Esc)  
- 📊 Progress bar during slideshow transitions  
- 📱 Responsive layout with hover effects  

---

🛠️ Tech Stack
HTML5 for structure

JavaScript (ES6) for interactivity
